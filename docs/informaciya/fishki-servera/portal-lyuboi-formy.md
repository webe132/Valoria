# Портал любой формы

На сервере снято ограничение на форму порталов в Ад - вы можете создавать порталы любой формы и размера!

<br>
**Примеры форм:**

1. Круглые порталы
2. Треугольные порталы
3. Порталы нестандартных размеров
4. Любые другие формы на ваш вкус
