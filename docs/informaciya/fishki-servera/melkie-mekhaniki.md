# Мелкие механики

### Восстановление коры

С помощью костной муки и `ПКМ` можно восстановить кору у обтёсанного дерева.

### Очистка земли

С помощью кисточки и `ПКМ` можно превратить подзол в дёрн.

### Ремонт наковальни

Повреждённую наковальню можно починить блоком железа. Возьмите блок железа в руку, зажмите `Shift` и кликните по наковальне `ПКМ`.

### Превращение обсидиана в плачущий

С помощью слезы гаста и `ПКМ` можно превратить обсидиан в плачущий.

### Поглаживание игроков

На сервере установлен плагин PatPat, который позволяет гладить других игроков.

### Снятые ограничения в наковальне

В наковальне больше не будет появляться надпись «Не слишком дорого» при дорогих скрещиваниях.

### Обработка дерева

Теперь камнерез может распиливать деревья также , как и с любой каменной породой.
