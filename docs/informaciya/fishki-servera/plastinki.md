# Пластинки

На нашем сервере доступна механика создания собственных музыкальных пластинок, позволяющая игрокам записывать и воспроизводить кастомную музыку прямо в игре.

### Основные возможности <a href="#osnovnye-vozmozhnosti" id="osnovnye-vozmozhnosti"></a>

С помощью специального аддона для PlasmoVoice вы можете:

* Записывать собственную музыку на пластинки
* Делиться своими творениями с другими игроками
* Воспроизводить записи через проигрыватель
